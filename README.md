# Colorguessing

A color memory game built with React, TypeScript, and Vite.

## Features

- Test your color memory skills
- HSL color matching gameplay
- Animated theme toggle (light/dark mode)
- Responsive design
- Score tracking

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Open your browser to the provided URL

## Build

To build for production: `npm run build`
